Bash — přejmenování souborů

parameter expansion, rename, mv a praktické příklady

1. Parameter expansion — ořezávání řetězců

Bash umí přímo v proměnné odstraňovat části řetězce bez externích příkazů. Klíčová syntaxe, která stojí za konstrukcí dir="${f%.cbz}":

SyntaxeCo děláPříklad pro f="Saga.vol1.cbz"
${f%.*}Odstraní nejkratší shodu od konceSaga.vol1
${f%%.*}Odstraní nejdelší shodu od konceSaga
${f#*.}Odstraní nejkratší shodu od začátkuvol1.cbz
${f##*.}Odstraní nejdelší shodu od začátkucbz

Mnemotechnická pomůcka: # je vlevo na klávesnici → maže od začátku, % je vpravo → maže od konce. Zdvojení (##, %%) = greedy.

Přesný pattern místo wildcards

Místo ${f%.*} (odstraní cokoliv od poslední tečky) lze zadat přesnou příponu:

# Odstraní přesně .cbz — bezpečnější, netkne se jiných přípon
dir="${f%.cbz}"

# Odstraní cokoliv od poslední tečky — .cbz, .zip, .pdf...
dir="${f%.*}"

2. Další parameter expansion triky

Nahrazování v řetězci

f="Muj Soubor (kopie).cbz"

# Nahradit první výskyt
echo "${f/ /_}"          # Muj_Soubor (kopie).cbz

# Nahradit všechny výskyty
echo "${f// /_}"         # Muj_Soubor_(kopie).cbz

# Odstranit znaky (nahradit za nic)
echo "${f//[()]/}"       # Muj Soubor kopie.cbz

Změna velikosti písmen (bash 4+)

f="Saga.CBZ"

echo "${f,,}"            # saga.cbz     (vše na malá)
echo "${f^^}"            # SAGA.CBZ     (vše na velká)
echo "${f,}"             # saga.CBZ     (první na malé)
echo "${f^}"             # Saga.CBZ     (první na velké)

Podřetězec

f="2024-03-15_backup.tar.gz"

echo "${f:0:10}"         # 2024-03-15   (prvních 10 znaků)
echo "${f:11}"           # backup.tar.gz (od 11. znaku)
echo "${f: -6}"          # tar.gz       (posledních 6, pozor na mezeru!)

3. Praktické příklady

CBZ do adresáře (původní skript)

#!/bin/bash
for f in *.cbz; do
  [ -f "$f" ] || continue
  dir="${f%.cbz}"
  mkdir -p "$dir"
  mv "$f" "$dir/"
done

Hromadná změna přípony

# .jpeg → .jpg
for f in *.jpeg; do
  mv "$f" "${f%.jpeg}.jpg"
done

Mezery na podtržítka

for f in *\ *; do
  mv "$f" "${f// /_}"
done

Přidat prefix/suffix

# Přidat datum jako prefix
for f in *.jpg; do
  mv "$f" "2024_${f}"
done

# Přidat _backup před příponu
for f in *.conf; do
  mv "$f" "${f%.conf}_backup.conf"
done

Lowercase celého jména

for f in *; do
  lower="${f,,}"
  [ "$f" != "$lower" ] && mv "$f" "$lower"
done

4. Příkaz rename

Pro složitější přejmenování existuje rename (Perl varianta, v Debianu apt install rename):

# .jpeg → .jpg
rename 's/\.jpeg$/.jpg/' *.jpeg

# Mezery na podtržítka
rename 'y/ /_/' *

# Lowercase
rename 'y/A-Z/a-z/' *

# Odstraní prefix "IMG_"
rename 's/^IMG_//' IMG_*

# Číslování: foto1.jpg, foto2.jpg...
i=1; for f in *.jpg; do mv "$f" "foto$((i++)).jpg"; done

5. Bezpečnostní tipy

  • Vždy uvozovky kolem proměnných: "$f", ne $f — soubory s mezerami jinak rozbijí skript
  • Testuj s echo nejdřív: echo mv "$f" "${f%.cbz}" — uvidíš co by se stalo
  • Guard na glob: [ -f "$f" ] || continue — pokud žádný soubor neexistuje, glob se neexpanduje a *.cbz zůstane jako literál
  • Pozor na přepsání: mv -n (no-clobber) nepřepíše existující soubor
  • Samba + symlinky: skript pouštěj přímo na serveru, ne přes /Volumes/Share/
