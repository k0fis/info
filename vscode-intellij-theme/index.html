VSCode / VSCodium — IntelliJ IDEA theme

JetBrains Mono font + IntelliJ IDEA Islands Theme — aby editor nevypadal jako VSCode
  • Theme: IntelliJ IDEA Islands Theme (OleksandrHavrysh)
  • Font: JetBrains Mono v2.304
  • Editor: VSCode i VSCodium (automaticka detekce)
  • OS: macOS
  • Skript: setup-intellij-theme.sh

Rychly start

Jednim prikazem — skript detekuje jestli mas code nebo codium a nastavi oba:

# Stahnout skript a spustit:
curl -fsSL https://info.kofis.eu/vscode-intellij-theme/setup-intellij-theme.sh | bash

Skript provede:

  1. Stahnuti a instalaci fontu JetBrains Mono do ~/Library/Fonts/
  2. Instalaci extensiony IntelliJ IDEA Islands Theme
  3. Nastaveni theme a fontu v settings.json

VSCodium je open-source build VSCode bez MS telemetrie. Pouziva Open VSX registry misto MS Marketplace. Pokud extension neni na Open VSX, skript ji stahne jako .vsix primo z marketplace.

Instalace VSCodium na macOS

brew install --cask vscodium

Rucni instalace

1. JetBrains Mono font

# Stahnout z https://www.jetbrains.com/lp/mono/
# nebo:
curl -fsSL -o /tmp/JBMono.zip \
  "https://github.com/JetBrains/JetBrainsMono/releases/download/v2.304/JetBrainsMono-2.304.zip"
unzip -qo /tmp/JBMono.zip -d /tmp/JBMono
cp /tmp/JBMono/fonts/ttf/JetBrainsMono-*.ttf ~/Library/Fonts/

2. Theme extension

# VSCode
code --install-extension OleksandrHavrysh.vscode-intellij-theme

# VSCodium (pokud je na Open VSX)
codium --install-extension OleksandrHavrysh.vscode-intellij-theme

# VSCodium (rucne pres VSIX)
# Stahnout .vsix z marketplace stranky, pak:
codium --install-extension cesta/k/theme.vsix

3. Settings

Otevrit Cmd+, → Open Settings (JSON) a pridat:

{
  "workbench.colorTheme": "IntelliJ IDEA Islands Dark",
  "editor.fontFamily": "JetBrains Mono, monospace",
  "editor.fontSize": 13,
  "editor.fontLigatures": true
}
